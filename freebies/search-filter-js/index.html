How to make a search and filter with pure JavaScript

Posted on: 5 July 2022.

What exactly is it? And why is it so special?

I was working on a project for a friend and she wanted a search filter on her site. Most active search filters require a database connection or XHR requests. That was a bit too heavy for me. All I wanted was a functionality that searched through the current page, something light and simple. I couldn't really find anything so I set about making one myself.

This code snippet is a few lines of codes and you don't need any other libraries or frameworks.

Here's how it works.

I've written this for newbies, so if you're advanced - you may get a little bored.

1. The search input.
In your html file, you have to paste this: 

<input type="search" class="js-search-field" placeholder="Type a word.."/>
If you're a beginner and do not know any JavaScript, make sure that the classname is the same. In JavaScript it finds 
.js-search-field
and has an EventListener attached to it. That eventlistener "scans" or "listens" for the users input from the keyboard.

If it turns out that in your project you can't paste that input - you can change the classname to the classname or identifier you may have to use in the JavaScript.

2. The search items.
If you're a beginner this is where it may get tricky . If you're using pure HTML and CSS like I am, then it's easy. The JavaScript code finds a classname in the HTML that has all the search entries you want to search through. 

 <li class="search-item">

# Typography

## Headings

### Guidelines
- H1 only in header.
- H2 for main sections.
- Default color: black.
- Default font-family: Bricolage Grotesque

_Including margins and responsive sizes._
- H1 `.site-title`:
    - font-size: 24px

- Small Text
`span.text__small`:
    - font-size: 12px
    - font-weight: 400

- Meta Data
_Used for portfolio items, for articles_
`.meta`:
    - font-family: var(--header-font);
    - font-size: 12px
    - font-weight: 400
    - display: block;
    - margin-bottom: 0px;
